# TeachingMaterials

这个仓库现在包含一个可直接离线运行的贪吃蛇小游戏示例。

## 运行方式

1. 直接用浏览器打开 `snake.html`。
2. 点击“开始游戏”或按空格键开始。
3. 使用方向键或 `W/A/S/D` 控制移动。
4. 按 `R` 可以随时重新开始。

## 文件说明

- `snake.html`：单文件版贪吃蛇小游戏，包含界面、逻辑、计分和本地最高分保存。
- `气候变化与健康数据可视化_两次课Slides.html`：原有教学幻灯片页面。
- `HourlyEnvironAndFlu_DLNM_Application.ipynb`：原有分析 notebook。
